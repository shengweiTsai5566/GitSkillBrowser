# 產品定義：內部 Agent 技能市集

## 1. 願景 (Vision)
建立一個公司內部的集中式「技能市集」（參考 [SkillsMP](https://skillsmp.com/) 模式），讓員工能夠輕鬆探索、分享與管理 AI Agent 技能。此平台將作為內部 AI 能力分享的樞紐，標準化 Agent 工具在團隊間的分發與使用方式。

## 2. 目標受眾 (Target Audience)
*   **員工 (開發者/使用者)：** 尋找特定的 Agent 能力以增強工作流程，或分享他們自己創建的工具。
*   **管理者：** 希望為團隊配備標準化、高效的 AI 工具。

## 3. 核心功能
*   **技能探索 (Skill Discovery)：** 基於 Web 的儲存庫，可瀏覽、搜尋和篩選 Agent 技能（支援 `SKILL.md`）。
    *   **雙語支援：** 預設顯示中文 (ZH)，並提供便捷的英文 (EN) 切換。
    *   **使用追蹤：** 即時追蹤每個技能的下載次數。
*   **雙重介面：**
    *   **Web UI：** 用於瀏覽、閱讀文件、手動上傳/下載。
    *   **REST API：** 用於自動化上傳、更新和下載技能（CI/CD 整合）。
*   **技能管理：**
    *   **所有權控制：** 使用者僅能修改或更新自己上傳的技能。
    *   **版本控制：** 支援透過 API/Web 更新現有技能。
*   **品質與安全：**
    *   **Schema 驗證：** 上傳時自動檢查 `SKILL.md` 和元數據結構的合規性。
    *   **安全掃描：** 自動預掃描秘密資訊（API Keys）和基本安全風險。

## 4. 執行模式 (Execution Model)
*   **本地安裝：** 平台作為分發中心。使用者將技能包下載到其本地環境（例如 `.agent/skills` 目錄）供其 Agent 使用。
