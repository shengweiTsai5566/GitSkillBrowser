# ⚔️ 新手村任務：從零開始成為 Skill Browser 的開發者

歡迎來到新手村！如果你不知道從哪裡開始，請跟著這張地圖走。
不用急著把書看完，我們**「做中學」 (Learn by Doing)**！

## Lv 1. 裝備整理 (環境準備) 🛡️
在出發打怪前，你需要先安裝這些工具：
1.  **VS Code**：這是我們寫程式的武器 (編輯器)。
2.  **Node.js**：這是讓網站跑起來的引擎。
3.  **Git**：這是時光機，讓你寫錯程式可以隨時後悔。

## Lv 2. 認識網頁三劍客 (HTML / CSS / JS) 🤺
*   **任務目標**：看得懂網頁上的標籤是什麼意思。
*   **HTML** 是骨架 (這是標題、那是按鈕)。
*   **CSS** 是衣服 (紅色、大字體)。
*   **JavaScript** 是動作 (點了會跳起來)。
*   **怎麼學？** 搜尋 `HTML CSS 1小時入門` 影片，看前 20 分鐘就好。

## Lv 3. 玩轉 React 與 Next.js (開始蓋房子) 🏠
這是最重要的一關！我們用的是 Next.js 14 (App Router)。
*   **任務目標**：試著修改首頁的一個文字。
*   **怎麼學？**
    1.  [Next.js 官方互動教學](https://nextjs.org/learn) (強烈推薦！像玩遊戲一樣)。
    2.  **實戰練習**：打開 `src/app/page.tsx`，把標題改成你的名字，存檔，看網頁變了沒！

## Lv 4. 資料庫探險 (Prisma) 🗺️
*   **任務目標**：查看現在資料庫裡有哪些技能。
*   **怎麼學？**
    *   不用寫程式！我們有魔法指令。
    *   **實戰練習**：
        1. 在 VS Code 下方的終端機 (Terminal) 輸入：`npx prisma studio`
        2. 瀏覽器會跳出一個視窗，你可以像操作 Excel 一樣直接查看、修改資料庫裡的資料！

## 💡 給初學者的超實用建議

1.  **不要從頭讀到尾**：程式碼成千上萬行，從頭讀你會睡著。請從「你想改的地方」開始讀。
2.  **善用 AI**：遇到看不懂的程式碼，複製貼給 AI，問它：「請用國小生聽得懂的話解釋這段在做什麼」。
3.  **大膽破壞**：反正我們有 Git 時光機，改壞了就還原，不要怕！

祝你冒險愉快！ 🚀
