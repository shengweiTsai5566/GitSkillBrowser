# 🗺️ 技術藏寶圖：Skill Browser 是怎麼運作的？

這份地圖是為了第一次來到這裡的冒險者準備的。
我們不要講太難的電腦名詞，我們用「**開一家技能圖書館**」來比喻！

## 1. 我們的櫃台與閱覽室 (前端 Frontend)
這裡就是**大家一進網站看到的地方**，漂亮的按鈕、排得很整齊的卡片都在這裡。

### **Next.js (超級館長)** 👔
*   **他是誰？** 這家圖書館的總負責人。
*   **他在做什麼？** 不管是客人要借書、要還書，還是要找廁所，全部都聽他指揮。他非常聰明，客人還沒開口，他就先把熱門的書準備好了（這叫做伺服器渲染）。
*   **你要學什麼？** 去了解 Next.js 的「頁面 (Page)」是怎麼變出來的。

### **React (樂高積木)** 🧱
*   **它是什麼？** 我們的書架、櫃台、借書按鈕，全部都是用 React 這個積木堆出來的。
*   **它能做什麼？** 我們寫好一個「按鈕積木」，就可以在整個網站到處用，不用每次都重新做一個。
*   **你要學什麼？** 學習怎麼把小積木 (Component) 組合成大城堡。

### **Tailwind CSS (神奇畫筆)** 🎨
*   **它是什麼？** 負責讓網站變漂亮的魔法。
*   **它能做什麼？** 以前我們要幫按鈕塗顏色很麻煩，現在用 Tailwind，我們只要對按鈕喊：「變藍色！變圓一點！」，它就自動變好了。
*   **你要學什麼？** 學習它的咒語（例如 `text-red-500` 就是把字變紅色）。

---

## 2. 我們的地下書庫 (後端 Backend & Database)
這裡是一般客人進不來的地方，用來存放所有資料和秘密。

### **PostgreSQL (超級筆記本)** 📓
*   **它是什麼？** 這是一個超級巨大、永遠寫不完的 Excel 表格。
*   **它記什麼？** 我們把所有的技能名稱、誰是作者、被下載了幾次，全部工整地寫在這裡面。它非常可靠，絕對不會弄丟資料。
*   **你要學什麼？** 知道什麼是「資料表 (Table)」就好，想像它就是一張 Excel 表。

### **Prisma (跑腿小弟)** 🏃
*   **他是誰？** 我們跟超級筆記本之間的翻譯官。
*   **他在做什麼？** 我們不用自己去翻那本厚厚的筆記本，我們只要跟 Prisma 說：「嘿！幫我找所有關於 PDF 的書」，Prisma 就會幫我們衝進去書庫找出來交給我們。
*   **你要學什麼？** 學習怎麼用 Prisma 寫出「幫我找書」的命令。

---

## 3. 進貨來源 (外部連線)

### **Gitea/GitHub (出版社)** 🏭
*   **它是什麼？** 我們的技能書本不是憑空出現的，都是從 Git 這個大出版社進貨的。
*   **怎麼運作？** 當我們按「同步」按鈕時，就是派卡車去 Git 出版社把最新的書載回來上架。

---

## 簡單總結 (一句話講完)

1.  用 **React** 做積木。
2.  用 **Tailwind** 上顏色。
3.  **Next.js** 把積木蓋成圖書館，並指揮大家做事。
4.  **Prisma** 幫我們跑腿，去 **PostgreSQL** 倉庫拿資料。

就這麼簡單！ 🎉
